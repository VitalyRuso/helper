# Demo: подготовка к cita previa

Это демонстрационный документ для проверки локального RAG в Spain Helper AI.

Перед визитом по cita previa обычно стоит проверить:

- официальный сайт органа, где назначена запись;
- адрес, дату и время приема;
- тип процедуры, выбранный при записи;
- список документов, указанный в официальной инструкции;
- наличие оригиналов, копий и переводов, если они прямо требуются.

Этот документ не устанавливает финальные юридические требования. Для подачи нужно сверяться с официальным источником конкретной процедуры, Extranjería, ayuntamiento или другим компетентным органом.
